# Rutinas Nutricionales

Aplicación de rutinas nutricionales diseñada para organizar dietas prescritas, planificación semanal y lista de la compra.

## Objetivo del proyecto

El objetivo de esta app es facilitar el seguimiento de rutinas nutricionales de forma clara, organizada y práctica.

La propuesta está pensada para casos en los que una persona necesita consultar pautas de alimentación, estructurar menús o preparar una lista de compra semanal sin depender de documentos dispersos o notas sueltas.

## Características principales

- Organización de rutinas nutricionales.
- Estructura orientada a dietas prescritas.
- Planificación semanal.
- Lista de la compra semanal.
- Interfaz ligera y orientada a uso cotidiano.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Estado del proyecto

Proyecto funcional en fase inicial de aprendizaje y mejora continua.

## Nota importante

Esta aplicación es una herramienta de organización y seguimiento. No sustituye el criterio de profesionales sanitarios ni debe utilizarse como diagnóstico o tratamiento médico.

## Aprendizajes trabajados

Con este proyecto se practican conceptos como:

- Organización de información nutricional en una interfaz web.
- Diseño de herramientas para casos de uso reales.
- Estructuración de rutinas semanales.
- Creación de listas funcionales para el usuario.
- Desarrollo web con HTML, CSS y JavaScript.
